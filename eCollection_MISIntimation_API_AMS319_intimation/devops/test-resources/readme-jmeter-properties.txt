JMeter Property Name													Mandatory			Description
ixc_api_endpoint_hostname												Y						IBM ACE HTTP Host Name (Pod Name / Service Route) where the client customized REST application is deployed.
ixc_api_endpoint_port_number										Y						IBM ACE HTTP Port where the client customized REST application is deployed. Default 7800.
ixc_api_base_uri																		Y						Base URI in the Swagger YAML REST descriptor of the deployed application. General format is /api/eCollection/mis/<Client Code>/<Business Scenario Name>
ixc_api_intimation_uri															Y						URI Path to Intimation and Remitter Validation operations.
ixc_api_refund_intimation_uri											Y						URI Path to Refund Intimation operation
ixc_api_client_organization_code										Y						Organization Code of the on-boarded ICICI Client / Customer.
ixc_api_client_code																Y						Client Code of the on-boarded ICICI Client / Customer.
ixc_api_subscription_product_name								Y						Subscribed Product Name. Default "eCollection"
ixc_api_subscription_business_scenario_name			Y						Subscribed Business Scenario Name.
ixc_api_virtual_account_number										Y						Client Virtual Account Number
ixc_api_client_deployment_build_id								Y						Deployment Build ID (generated by Jenkins)
ixc_api_client_deployment_target_env							Y						Deployment Target Environment
ixc_api_client_deployment_validation_userId				Y						User ID of the user that initiated the build / deployment through the Customer On-Boarding Portal. Default "ixc-system"
ixc_concurrent_user_count												N						Concurrent users / threads to run. Default 1.
ixc_user_loading_seconds													N						User / Thread ramp up time. Default 1.
ixc_record_count_intimation												Y						No of records from the test data file to use for the test run. Default 1.
ixc_record_count_refund_intimation								Y						No of records from the test data file to use for the test run. Default 1.
ixc_test_data_filename_intimation									Y						Test data file name for Intimation & Remitter Validation test runs.
ixc_test_data_filename_refund_intimation					Y						Test data file name for Refund Intimation test runs.
ixc_test_report_name_intimation									Y						Test report file name for Intimation & Remitter Validation test runs.
ixc_test_report_name_refund_intimation						Y						Test report file name for Refund Intimation test runs.
ixc_api_elasticsearch_endpoint_uri_operational	Y	Endpoint to lookup for IBM ACE Operational Events.	Operational Events are IBM ACE Monitoring Events but with business data cleansed.
ixc_api_elasticsearch_endpoint_uri_business	Y	Endpoint to lookup for IBM ACE Business Events.	
ixc_api_elasticsearch_endpoint_uri_jmeter_testreport	Y	Endpoint to publish JMeter Test Reports	
ixc_api_client_project_id	Y	Client Project ID (Portal API)	
ixc_api_build_git_commit_id	Y	Git repository commit ID to track the code snapshot when performing a build verification & subsequent deployment.
